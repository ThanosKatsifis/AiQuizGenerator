

# 🧠 SmartyPants AI: The Smartphone Quizzer

Ever wondered if you're actually smarter than a fifth grader—or at least more tech-savvy than an 18-year-old? **SmartyPants AI** is a Python-powered quiz application that uses GPT-3.5 Turbo to generate "hard-mode" questions about smartphones in real-time.

It’s sleek, it’s purple (thanks to the Yaru theme), and it’s ready to judge your tech knowledge.

---

## 🚀 Features

* **Infinite Questions:** Powered by OpenAI, so you'll never see the same question twice (unless the AI gets nostalgic).
* **Slick UI:** Built with `Tkinter` and `ttkthemes` for that modern Linux-style aesthetic.
* **Real-time Scoring:** Watch your score climb or stay embarrassingly low as you answer.
* **Hard Mode Only:** Designed specifically for students who think they know everything about their devices.

---

## 🛠️ Built With

* **Python 3.x** - The backbone.
* **OpenAI API** - The "Brain" behind the questions.
* **Tkinter / ttkthemes** - For a GUI that doesn't look like it was made in 1995.

---

## 📦 Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/smartypants-ai.git
    cd smartypants-ai
    ```

2.  **Install the dependencies:**
    ```bash
    pip install openai ttkthemes
    ```

3.  **Set up your API Key:**
    Replace the `api_key` string in the code with your own OpenAI secret key. 
    *(Note: Never share your key publicly!)*

4.  **Run the app:**
    ```bash
    python main.py
    ```

---

## 🎮 How to Play

1.  The app will automatically fetch a tricky "Yes/No" question about smartphones.
2.  Click **Yes** or **No**.
3.  The app instantly tells you if you're a genius or if you need to go back to tech school.
4.  Your score updates automatically. Try to get to 10 without failing!

---

## ⚠️ Disclaimer

This app is set to "Hard" difficulty. If you get a question wrong, don't blame the code—blame the AI! (Or maybe just study more).
